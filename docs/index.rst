Welcome to Attendance Reporter's documentation!
================================================

Attendance Reporter is a PHP software program used at Andrews University's
mathematics and physics departments to track student participation in
weekly colloquiums (eigen*) and small group learning (SGL) sessions.

Any questions not answered in these docs can be referred to Jonathan Wheeler
at <jonathan.m.wheeler@gmail.com>
